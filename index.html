Main Features

  • Advanced moderation commands: warn, mute, unmute, kick, ban, unban
  • Voice moderation: vcmute, vcunmute, vcdeafen, vcundeafen, vckick
  • Music system with YouTube support (play, pause, resume, stop)
  • Auto-disconnect after inactivity with text-channel notification
  • Owner-configurable logging system
  • Professional /help command and clean presence

Advanced Features

  • Full warning system with history tracking
  • /warnings command showing all warns for a user
  • Logs include: who used the command, target user, reason, and timestamps
  • Automatic unmute/undeafen after timer expires
  • Automatic logging of all moderation and voice actions
  • Safe and reliable timeout-based moderation
